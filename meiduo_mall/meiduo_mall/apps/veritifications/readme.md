### 实现短信验证码


**访问方式**： GET `/sms_codes/(?P1[3-9]\d{9})/`

**请求参数**： 路径参数与查询字符串参数

| 参数     | 类型   | 是否必须 | 说明   |
| ------ | ---- | ---- | ---- |
| mobile | str  | 是    | 手机号  |

**返回数据**： JSON

| 返回值     | 类型   | 是否必传 | 说明      |
| ------- | ---- | ---- | ------- |
| message | str  | 否    | OK，发送成功 |
